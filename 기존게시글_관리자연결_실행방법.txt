기존 게시글 8개 관리자 CMS 연결 방법

1. 먼저 Supabase SQL Editor에서 새 쿼리를 엽니다.
2. SUPABASE_기존게시글8개_CMS이전.sql 파일 전체를 복사해 실행합니다.
3. Success. No rows returned가 나오면 정상입니다.
4. 그 다음 v9.3 파일 전체를 GitHub 저장소에 덮어씁니다.
5. 관리자 > 시공사례 관리에서 기존 게시글 8개가 목록에 나타납니다.

연결 후 가능한 작업
- 제목, 카드 설명, 작업 내용, 지역, 상세 설명 수정
- 새 사진 추가
- 기존 사진 삭제
- 공개/비공개 전환
- 게시글 완전 삭제

인덱스 대표 시공사례는 각 카테고리에서 가장 최근 게시글 1개를 자동 표시합니다.
앞으로 새 게시글을 등록하면 해당 카테고리의 인덱스 대표 카드도 자동으로 최신 글로 바뀝니다.

기존 post-*.html 파일은 호환성을 위해 남겨두지만, 새 카드들은 공통 project-detail.html을 사용합니다.
