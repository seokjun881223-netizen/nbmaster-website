난방 마스터 시공사례 관리자 CMS 설치 안내

이번 버전에서 새로 생긴 기능
1. 관리자 로그인 후 '시공사례 관리' 진입
2. 카테고리, 제목, 설명, 지역, 사진 여러 장 입력
3. 등록 버튼을 누르면 해당 카테고리와 전체 시공사례에 카드 자동 생성
4. 첫 번째 사진은 카드 대표사진으로 자동 사용
5. 상세 게시글은 선택한 사진 수만큼 슬라이드 자동 생성
6. 관리자 화면에서 수정, 비공개, 삭제 가능
7. GitHub 코드를 다시 편집할 필요 없음

최초 1회 설정
1. Supabase에 로그인
2. SQL Editor > New query
3. 이 폴더의 'SUPABASE_시공사례_CMS_설치.sql' 전체 복사
4. 붙여넣고 Run
5. Success. No rows returned 확인
6. 이 프로젝트 전체를 GitHub 저장소에 업로드하여 기존 파일 교체

관리자 접속 주소
https://nbmaster.co.kr/admin-login.html

로그인 후
- 문의 관리
- 시공사례 관리
두 메뉴가 표시됩니다.

중요
- 기존 정적 시공사례 8개는 그대로 유지됩니다.
- 관리자 화면에서 새로 등록한 게시글만 Supabase에서 자동 추가됩니다.
- 사진 한 장당 최대 15MB입니다.
- 대표사진은 업로드할 때 선택한 첫 번째 사진입니다.
