난방 마스터 v6.4

수정 내용
- 홈페이지 타이틀: NB MASTER → 난방 마스터
- ABOUT US: NB MASTER → 난방마스터
- WHY 문구: 왜 NB MASTER를 선택할까요? → 왜 난방 마스터를 선택할까요?
- favicon.ico 및 모바일 아이콘 추가
- CSS에서 참조하던 누락 hero.jpg 파일 추가
- 모든 HTML/CSS/JS/이미지 경로 검증
- VS Code Live Server를 모바일 접속 가능한 0.0.0.0:5500으로 설정

모바일 접속 필수 사항
1. 휴대폰에서 127.0.0.1로 접속하면 안 됩니다.
   127.0.0.1은 휴대폰 자기 자신을 의미합니다.
2. PC와 휴대폰을 같은 Wi-Fi에 연결하세요.
3. Windows에서 ipconfig를 실행해 PC의 IPv4 주소를 확인하세요.
4. 휴대폰에서 아래 형식으로 접속하세요.
   http://PC의IPv4주소:5500/index.html
   예: http://172.30.1.83:5500/index.html
5. Windows 방화벽 창이 뜨면 개인 네트워크 접근을 허용하세요.

실행 방법
A. VS Code에서 index.html 우클릭 → Open with Live Server
B. 모바일테스트_서버실행.bat 더블클릭
